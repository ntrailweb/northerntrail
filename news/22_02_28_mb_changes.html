Changes To Requirements 2021/2022

A number of merit badges had changes in their requirements last year and others are changing this year. A Scout who started a merit badge using the “prior” requirements may continue with those requirements; while Scouts beginning a merit badge should use the most current set of requirements. For more information see Guide to Advancement, Section 7.0.4.3, What to do When Requirements Change (https://filestore.scouting.org/filestore/pdf/33088.pdf). The 2021 changes are currently reflected in Scoutbook while the 2022 changes will be posted shortly. All current merit badge requirements can be found here: https://www.scouting.org/programs/scouts-bsa/advancement-and-awards/merit-badges/.
